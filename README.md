# Portfolio

## Overview

Welcome to the repo!

## Running Locally

```bash
$ git clone https://github.com/FranciscoMoretti/portfolio.git
$ cd portfolio
$ yarn
$ yarn dev
```

Create a `.env.local` file similar to `.env`.

```
FIREBASE_PRIVATE_KEY=
FIREBASE_CLIENT_EMAIL=
FIREBASE_PROJECT_ID=
FIREBASE_DATABASE_URL=
```
